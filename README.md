Explain the difference between RGB and CMYK color mode. Also calculate;

If a photograph that is 15 inches wide and is 8 inches in height, and

If it's to be displayed at 300 dpi resolution,

What is its Width and Height supposed to be in PIXELS?

Ans:
RGB Color Mode
- RGB stands for **Red, Green, Blue**.
- It is used for **digital screens** such as computers, mobiles, TVs, and websites.
- Colors are created by mixing light.
- When all three colors are combined at full intensity, it creates **white**.

 CMYK Color Mode
- CMYK stands for **Cyan, Magenta, Yellow, Key (Black)**.
- It is used for **printing purposes** such as posters, books, magazines, and banners.
- Colors are created by mixing ink.
- When all colors are combined, it creates dark shades, so black ink is added separately.

 Main Difference Between RGB and CMYK

| RGB | CMYK |
|-----|------|
| Used for screens | Used for printing |
| Based on light colors | Based on ink colors |
| Produces brighter colors | Produces print-friendly colors |
| Red, Green, Blue | Cyan, Magenta, Yellow, Black |

---

 Pixel Calculation

Given:
- Width = **15 inches**
- Height = **8 inches**
- Resolution = **300 dpi**

 Formula:
Pixels = Inches × DPI**
Width in Pixels:
15 × 300 = **4500 pixels**

 Height in Pixels:
8 × 300 = **2400 pixels**

 Final Answer:
- **Width = 4500 pixels**
- **Height = 2400 pixels**

 Image Size:
**4500 × 2400 pixels**

2. List out and explain the features of the four major types of raster image file extensions. Point out the image file extension that supports both transparency as well as animation. Also calculate;

What is the file size of a 24 bit color image of the dimension 1000px X 500px in MB ?


Ans: 
Four Major Types of Raster Image File Extensions

Raster images are made up of tiny pixels. The four major raster image file extensions are:

 1. JPEG / JPG (.jpg / .jpeg)
- Stands for **Joint Photographic Experts Group**.
- Commonly used for photographs and web images.
- Supports millions of colors.
- Uses **lossy compression**, so some image quality is reduced.
- Small file size.

 2. PNG (.png)
- Stands for **Portable Network Graphics**.
- Used for web graphics, logos, and transparent backgrounds.
- Uses **lossless compression**, so quality remains high.
- Supports **transparency**.

3. GIF (.gif)
- Stands for **Graphics Interchange Format**.
- Supports only 256 colors.
- Used for simple graphics and web animations.
- Supports **transparency**.
- Supports **animation**.

 4. BMP (.bmp)
- Stands for **Bitmap Image File**.
- Developed by Microsoft.
- Stores image data without much compression.
- High image quality but large file size.

---

File Extension Supporting Both Transparency and Animation

The image file extension that supports both **transparency** and **animation** is:

*GIF (.gif)**

---

File Size Calculation

 Given:
- Dimension = **1000 px × 500 px**
- Color Depth = **24 bit**

Formula:
**File Size = Width × Height × Bit Depth ÷ 8**

 Step 1: Total Pixels
1000 × 500 = **500,000 pixels**

 Step 2: Convert Bits to Bytes
500,000 × 24 = **12,000,000 bits**

12,000,000 ÷ 8 = **1,500,000 bytes**

Step 3: Convert to MB
1,500,000 ÷ 1,048,576 = **1.43 MB** (approx)

---

 Final Answer

**File Size = 1.43 MB (approximately)**
